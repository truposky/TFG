\subsection{Objetivos}
El objetivo es crear un entorno de pruebas para poder probar algoritmos colaborativos y de coordinación u otros en el cual sea necesario robots móviles como el cual se denomina robotario, el entorno está basado en los trabajos anteriormente mencionados de Duckietown y Robotarium. Para poder llevarlo acabo se necesita cumplir unos oobjetivos
\subsubsection{{\large objetivos específicos }}
\begin{itemize}
	\item Construir y diseñar un robot capaz de realizar los algoritmos requeridos por el robotario.
	\item Diseñar un controlador de la velocidad del robot que permita realizar los algoritmos de manera correcta.
	\item Configurar la comunicación de los robots, para poder comunicarse con otros robots y un servidor.
	
	\item Diseñar y configurar una red inalámbrica para la comunicación entre los robots y un servidor.
	\item Crear y configurar un servidor que reciba la posición de los robots y se comunique con los diferentes robots del robotario.
	\item Construir y diseñar un sistema de localización, que identifique y localice a los robots involucrados en los experimentos.
	\item (duda)Llevar a cabo un experimento para demostrar el correcto funcionamiento del robotario.
\end{itemize}
